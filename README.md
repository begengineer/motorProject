# motorProject

# モーター制御学習用コード
組み込みC言語の学習のためのにDCモーター制御プログラムを作ってみました。ArduinoMEGAベースで作成しています。

# 想定
このプログラムは、一定間隔で上下の動作をするモーター制御を想定しています。
制御パラメータを設定する状態、モーターを動作させる状態、一時停止をする状態、条件を満たして終了する状態の遷移をします。

以下実装内容
- 基本的なモーターの制御関数
- 基本的なスイッチの制御関数
- 割り込みによるモーター動作変更
- 関数ポインタテーブルを用いた状態遷移処理
- 関数ポインタを用いたコールバック関数処理

# 工夫ポイント
- モーターとスイッチをほかのプロジェクトでも利用できるようなコード構成にしました。
